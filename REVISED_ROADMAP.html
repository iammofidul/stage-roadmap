üéØ Initiative 4: Engagement Orchestration (NEW)

The KEY new piece: Coordinate existing features to work together instead of competing

Trial User Journey (Orchestrated)

Day 0: Trial Activation
Fire trial_activated event ‚Üí Show Content Onboarding (Medal System)
‚Üì
Day 0-1: Medal Unlocking
User unlocks 2-3 medals ‚Üí Track unlock speed
‚Üì
Day 1: First Engagement Check
If <2 videos watched ‚Üí Show Commitment Boost
If >2 videos ‚Üí Offer Special Access
‚Üì
Day 2: Paywall Moment
User returns to paywall ‚Üí 4th back press ‚Üí Spin the Wheel
OR Special Access offer
‚Üì
Day 3-5: Retention Phase
Daily push notification (personalized)
Discovery Boost if inactive >24h
‚Üì
Day 6-7: Pre-Expiry
Trial expiring soon ‚Üí Resume Trial nudge
Last chance Special Access unlock

What This Orchestrator Does:

  • Prevents overlapping nudges (no boost + spin wheel at same time)
  • Sequences features based on user behavior (medals ‚Üí special access ‚Üí spin)
  • Adapts to engagement level (high engagement = special access, low = boost)
  • Tracks cross-feature journey (medal_unlock ‚Üí special_access_claim ‚Üí payment)
  • Manages feature cooldowns (don't show boost if shown <6h ago)
