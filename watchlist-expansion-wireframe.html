User Flow Summary

  • Adding to List: User browses content ‚Üí Taps ‚ûï on any card ‚Üí Added to My List (‚úì shown)
  • Viewing List: User taps "My List" in navigation ‚Üí Sees all watchlisted content with badge counter
  • Home Visibility: "My List" row appears at top of home with all watchlisted items
  • 48h Reminder: Content added but not watched ‚Üí User receives "Don't forget" notification ‚Üí Taps "Watch Now"
  • Weekly Digest: New episodes added to watchlisted shows ‚Üí User receives weekly summary ‚Üí Taps "View My List"

Technical Requirements

  • Quick add button component on all content cards (home, search, details, player exit)
  • My List row widget on home screen (always visible if list has items)
  • Dedicated My List screen accessible from bottom navigation
  • Badge counter component showing list count (updates in real-time)
  • Background service for 48h reminder notifications (local scheduling)
  • Weekly digest notification (server-side cron job, Monday 9 AM)
  • Deep linking for "Watch Now" actions
  • Analytics: watchlist_add, watchlist_remove, watchlist_reminder_sent, watchlist_reminder_clicked, weekly_digest_sent, weekly_digest_clicked

Expected Impact

  • Watchlist Adds: 2.91% ‚Üí 8% (+5pp, +3,400 adds/month)
  • Watchlist ‚Üí Playback: 40% ‚Üí 60% (+20pp from notifications)
  • Engagement Lift: Exit intent popup gives 3.8x lift (10.61% vs 2.91%)
  • D7 Retention: Users with watchlists have 20% higher retention
